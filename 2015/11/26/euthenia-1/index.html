Experiment procedure

login ovs

$ ssh pica8-2

check if ovs is ready

$ ovs-vsctl show

set ovs if needed

??? $ ovs-vsctl br0 tcp:ip:port # ovs-vsctl br0 tcp:30.0.0.102:9000

add flows

$ ovs-vsctl add－flow [switch] [args...] # ovs-ofctl add－flow br0 action=output:controller

show flow status

$ ovs-ofctl dump-flows [switch] # ovs-ofctl dump-flows br0

start server

$ sudo ./eTester eth2
